# TER-Picopatt – branche `test`

Cette branche sert de zone de test.  
Elle regroupe les essais, prototypes et versions de travail qui ne sont pas encore stabilisés pour être intégrés dans les branches `Notebook`.

## À quoi sert cette branche

On y met :

- versions intermédiaires de notebooks ou scripts avant consolidation
- sorties temporaires (figures de debug, exports incomplets)
